# InnoMath

Innomath is an application designed to help visually and hearing impaired primary school students with their mathematics lessons. It is designed for the visually and hearing impaired.


### Architecture
- MVVM Design Pattern 
    
   
## Used in the project

- Firebase
- WorkManager
- Coroutines
- Flow
- Retrofit
- Room
- DataStore
